# :bank: Bank Management System

A simple Python project for managing bank accounts.  
This program is designed to practice lists, dictionaries, loops, and conditional statements in Python, while simulating real-world banking operations.

---

## :sparkles: Features / ویژگی‌ها

- Create New Account / ایجاد حساب جدید
  - Enter account holder name
  - Enter initial balance

- Deposit Money / واریز وجه
  - Add money to a specific account

- Withdraw Money / برداشت وجه
  - Withdraw money if sufficient balance exists

- Check Balance / نمایش تراز حساب
  - Check the current balance of any account

- Show All Accounts / نمایش همه حساب‌ها
  - List all accounts with balances

- Exit / خروج
  - Exit the program safely
